🧠 MVP – Machine Learning & Analytics: Previsão de Risco de AVC

Este projeto apresenta um modelo de Machine Learning para prever o risco de Acidente Vascular Cerebral (AVC) com base em características clínicas e demográficas dos pacientes.

O trabalho inclui todas as etapas de um projeto de ciência de dados: carregamento e exploração dos dados (EDA), tratamento de valores ausentes, pré-processamento, construção de pipelines, treinamento e comparação de diferentes algoritmos de classificação, otimização de hiperparâmetros e avaliação dos modelos utilizando métricas como Recall, Precisão, F1-score, ROC-AUC e matriz de confusão.

Como o conjunto de dados é desbalanceado, o projeto prioriza métricas capazes de identificar corretamente pacientes com maior risco de AVC, discutindo os impactos dessa escolha na aplicação prática.
